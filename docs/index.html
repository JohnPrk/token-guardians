토큰 판다

데스크톱 한구석에서 너의 Claude·Gemini 토큰 잔량을 지켜봐주는 작은 판다. 5분 캐시 골든타임을 놓치지 않게 신호를 보내요.

macOS .dmg 다운로드 Windows .exe 다운로드

macOS 11+ (Apple Silicon) · Windows 10+ x64 · 무료 · MIT

왜 만들었나

Claude의 유일한 단점은 토큰 한도예요.

질문에 답을 받고 5분 이내에 다음 대화를 이어가면, Claude는 기존 내용을 캐시해두었다가 토큰 소모량을 획기적으로 아껴줍니다. 하지만 다른 업무를 하다 보면 이 짧은 '골든타임'을 놓치기 쉬워요. 이제 화면 속 작은 판다가 그 타이밍을 놓치지 않게 신호를 보냅니다.

💡 마지막 요청에서 5분 안에 다음 요청을 보내면 그 직전까지의 컨텍스트가 캐시 히트로 처리돼 토큰 사용량이 1/10로 줄어듭니다.

설정은 이렇게

처음 한 번만 등록하면 30초마다 자동으로 갱신돼요.

1

캐릭터 고르기

데스크톱 모서리에 앉을 친구를 고릅니다 (판다 · 고양이 · 햄스터). 나중에 설정에서 언제든 변경 가능.

2

서비스 선택 — Claude 또는 Gemini

새 계정 추가에서 Claude 또는 Gemini 탭을 고릅니다. 두 서비스 모두 세션 쿠키 한 줄로 연동돼요.

3

쿠키 붙여넣기

안내대로 브라우저 DevTools → Network 탭에서 cookie: 한 줄을 복사해 붙여넣으면 끝. 테스트 버튼으로 바로 확인할 수 있어요. (Claude는 자동 가져오기도 지원)

어떤 기능이 있나

메뉴바에 살면서 토큰을 대신 지켜봐요.

📊

메뉴바 라이브 표시

5시간 / 주간 토큰 잔량을 메뉴바에 실시간으로 표시.

⏰

5분 캐시 타이머

질문을 던지는 순간 카운트다운이 시작되고, 만료 전 캐릭터가 흔들리며 신호.

🔢

캐시 Hit/Miss 추적

매 응답의 캐시 성공·실패 수치를 직접 확인해 토큰 절약 효율 관리.

🐼

8단계 캐릭터 상태

잔량에 따라 Full부터 Sleepy까지 외형·감정·동작이 변화.

🖱️

자유로운 배치

화면을 가리지 않고 클릭 앤 드래그로 어디든 이동. 빈 공간 클릭은 통과.

👥

멀티 계정 · 멀티 서비스

Claude·Gemini 계정을 여러 개 등록하고 카드 클릭으로 즉시 전환.

계정 전환 · 멀티 서비스

Claude와 Gemini를 한 펫에서 오가며 봐요.

계정을 여러 개 등록해두고 설정의 계정 카드를 클릭하면 활성 계정이 바뀌고, 메뉴바 잔량·펫 캐릭터·폴링 자격증명이 한 번에 그 계정으로 전환돼요. 서비스마다 인증 방식이 달라도 같은 "쿠키 한 줄" UX로 통일했습니다.

새로고침 · 자동 업데이트

한 번 켜두면 알아서 최신을 유지해요.

🔄

30초 자동 폴링

등록한 쿠키로 30초마다 잔량을 자동 갱신. 펫 우클릭으로 즉시 새로고침도 가능.

🔌

연결 끊김 자동 감지

쿠키가 만료되면 판다가 '연결 실패' 표지판을 들고, 설정 창이 자동으로 열려 새 쿠키를 요청. (Gemini는 회전 토큰을 자동 갱신해 오래 켜둬도 유지)

🆕

앱 자동 업데이트

새 버전이 나오면 트레이 헤더에 🆕 마커가 뜨고, 클릭 한 번으로 최신 버전을 받아 설치.
